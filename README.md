# bin-publish

A number of convenient scripts.

Install:
```
$ cd ~
$ git clone https://github.com/jhansen2015/bin-publish.git bin-publish
...

$ chmod +x bin-publish/*.sh

```

Then, add the `~/bin-publish` directory to your path in `.bash_profile` or `.bashrc`.
